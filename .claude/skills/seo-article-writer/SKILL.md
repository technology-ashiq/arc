# SEO Article Writer

Given a topic/keyword, produce a publish-ready article.

## Inputs
- Primary keyword (required), target audience, word count (default 1200–1500).

## Workflow
1. Derive 1 primary + 4–6 secondary keywords and the search intent.
2. Outline: H1, 5–8 H2s, FAQ section.
3. Draft with the keyword in the H1, first 100 words, one H2, and the meta description.
4. Add: meta title (<60 chars), meta description (<155 chars), slug, internal-link suggestions.
5. Short paragraphs, plain language, no keyword stuffing.

## Output
Markdown article + a frontmatter block (title, description, slug, keywords).
