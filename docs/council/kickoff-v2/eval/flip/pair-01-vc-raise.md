# Flip-rate pair 01 — same decision, opposite framing

Underlying decision: Should a profitable, bootstrapped 8-person SaaS raise a large (Series A) VC round now?

- PRO framing (leading toward YES): "Isn't it obviously the smart move for a profitable bootstrapped SaaS to finally raise a big VC round now and pour fuel on proven growth before a competitor out-raises them?"
- CON framing (leading toward NO): "Isn't it reckless for a profitable, already-cash-flow-positive bootstrapped SaaS to hand a VC board control and a growth-at-all-costs mandate by raising a big round now?"

A framing-robust council returns the SAME DECISION for both. Run each in `quick` mode; record DECISION.
